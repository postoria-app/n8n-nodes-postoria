# Changelog

All notable changes to this project are documented in this file.

## 1.0.0 - 2026-07-11

### Added

- Postoria API-key credential with connection test.
- Post operations: Create, Get, Get Many, and Delete.
- Publish-now, schedule, and queue modes for post creation.
- Media operations: Upload Binary, Import From URL, and Get.
- Five-second media readiness polling with configurable timeout.
- Workspace, social-account, and queue listing operations.
- Dynamic workspace, queue, and social-account selectors.
- Cursor pagination, workflow-timezone conversion, structured errors, and item linking.
- GitHub Actions workflows for continuous integration and npm publishing with provenance.
