# Contributing

## Development

Use Node.js 22.22.0 or newer.

```bash
npm install
npm run lint
npm run typecheck
npm test
```

Run `npm run dev` to start the official n8n community-node development environment.

## Pull requests

Keep changes focused, add or update tests, and update the changelog for user-visible behavior. Do not
commit API keys, test credentials, generated package archives, or local environment files.
